---
title: 'Evaluator Stats'
description: 'Get statistics and performance metrics for an evaluator'
---

## Endpoint

<ParamField path="method" type="string" default="GET">
  GET
</ParamField>

<ParamField path="endpoint" type="string">
  `/v1/evaluator/{evaluatorId}/stats`
</ParamField>

## Authentication

This endpoint requires API key authentication. Include your API key in the request headers:

```bash
Authorization: Bearer YOUR_API_KEY
```

## Path Parameters

<ParamField path="evaluatorId" type="string" required>
  The unique identifier of the evaluator
</ParamField>

## Response

<ResponseField name="data" type="object">
  Evaluator statistics
  <ResponseField name="averageScore" type="number">
    Average score across all evaluations
  </ResponseField>
  <ResponseField name="totalUses" type="number">
    Total number of times this evaluator has been used
  </ResponseField>
  <ResponseField name="recentTrend" type="string">
    Recent trend direction: "up", "down", or "stable"
  </ResponseField>
  <ResponseField name="scoreDistribution" type="array">
    Distribution of scores across ranges
    <ResponseField name="range" type="string">
      Score range (e.g., "0-20", "20-40")
    </ResponseField>
    <ResponseField name="count" type="number">
      Number of evaluations in this range
    </ResponseField>
  </ResponseField>
  <ResponseField name="timeSeriesData" type="array">
    Historical data over time
    <ResponseField name="date" type="string">
      Date in ISO format
    </ResponseField>
    <ResponseField name="value" type="number">
      Average score for that date
    </ResponseField>
  </ResponseField>
</ResponseField>

<ResponseField name="error" type="string | null">
  Error message if the request failed, null otherwise
</ResponseField>

## Example Request

```bash
curl -X GET https://api.helicone.ai/v1/evaluator/eval_abc123/stats \
  -H "Authorization: Bearer YOUR_API_KEY"
```

- Monitor evaluator performance over time
- Identify trends in evaluation scores
- Understand score distribution patterns
